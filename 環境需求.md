# IPCamClockSync 環境需求

## 最小環境
- Windows 11 Pro x64 一台。
- Windows Server 2022 一台。
- ONVIF 攝影機至少 3 台（至少 2 種型號）。
- .NET 8 Runtime。
- 可控內網（DHCP 與固定 IP 可測）。

## 建議完整環境
- 控制端 16GB RAM 以上。
- 攝影機 10 台以上，含 1 台離線模擬節點。
- 網路條件：正常、50-150ms 延遲、1%-5% 丟包、跨子網。
- 權限測試：Administrator、Standard User、LocalService。

## 工具建議
- 測試框架：xUnit + FluentAssertions。
- 覆蓋率：Coverlet（Core 模組目標 70% 以上）。
- CLI 驗證：輸出快照 + 回傳碼檢查。
