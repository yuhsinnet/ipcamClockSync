# IPCamClockSync 計劃

## 目標
以 .NET 8 開發 C# 工具，透過 ONVIF 掃描攝影機、單次校時與 NTP 設定，並提供 CLI 與 Console GUI。NTP Server 採獨立程式並支援 Windows Service。

## 里程碑
1. 建立專案骨架（Core、CLI、NtpServer、Tests）。
2. 完成設定/清單資料層（YAML/JSON）。
3. 完成單次校時與 NTP 推送流程。
4. 完成 NTP Service 安裝與防火牆規則自動化。
5. 補齊 Console GUI 與日誌分流。

## 既定決策
- NTP 架構：完全分離（GUI 控制 + NTP 獨立服務）。
- Service 預設：LocalService + Automatic (Delayed Start)。
- 防火牆：自動建立 UDP 123 規則。
- 日誌：JSONL、channel 分流、correlationId、輪替保留。
