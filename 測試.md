# IPCamClockSync 測試

## MVP 必測
- 掃描、保存、單次校時、NTP 推送可完整走通。
- Service 可 install/start/stop/restart/status。
- install 後為 Automatic (Delayed Start)。
- 防火牆規則可建立與查詢。
- 10 台設備連跑 3 輪無致命錯誤。

## 上線前必測
- Windows 10/11/Server 相容。
- Administrator/Standard User/LocalService 權限矩陣。
- 50/200 台壓力測試與 24 小時穩定性測試。
- 日誌輪替與 correlationId 追蹤完整。
- P0/P1 缺陷為 0。

## 單元測試重點
- 設定與清單讀寫驗證。
- ONVIF 掃描結果解析與逾時。
- 批次校時每台重讀系統時間。
- CLI 命令分群與錯誤碼。
- 服務/防火牆控制器邏輯。
